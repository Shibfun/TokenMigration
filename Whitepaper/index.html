Introduction

Woofswap Token Factory is a decentralized platform built on the Shibarium blockchain, enabling users to create, manage, and wrap tokens seamlessly. Leveraging Shibarium's low-cost and high-speed infrastructure, Woofswap empowers developers, communities, and businesses to tokenize assets with ease.

Our mission is to democratize token creation, making it accessible to anyone with a MetaMask wallet. The platform supports wrapping existing ERC-20 tokens into new tokens with customizable parameters, fostering innovation in DeFi, gaming, and community-driven projects.

Technical Architecture

The Woofswap Token Factory consists of a smart contract deployed on Shibarium (address: 0xE930a70e6055Af321Cc948DB9Ded5b9df08b5a8A) and a user-friendly frontend interface.

Smart Contract

  • Token Factory Contract: Allows users to create new ERC-20 tokens by specifying name, symbol, max supply, and underlying token address.
  • Token Contract: Manages wrapped tokens with deposit (mint) and withdraw (burn) functions, maintaining a 1:1 peg with the underlying token.
  • Query Functions: Supports querying tokens by owner and retrieving the latest 10 created tokens.

Frontend Interface

  • Create Token: Users input token details and deploy new tokens via MetaMask.
  • Manage Token: Deposit or withdraw tokens, with real-time balance updates.
  • Token Info: Displays token metadata (name, symbol, max supply, etc.) by entering a token address.
  • Created Tokens: Lists the latest 10 tokens created by the user, with copyable addresses.

Network Integration

Built on Shibarium (Chain ID: 109), Woofswap uses MetaMask for secure wallet connections. The platform ensures low transaction fees and fast confirmations, ideal for tokenization.

Key Features

  • Token Creation: Deploy ERC-20 tokens with customizable name, symbol, and max supply.
  • Deposit/Withdraw: Wrap and unwrap tokens with a 1:1 peg, supported by percentage-based input (25%, 50%, 75%, MAX).
  • Token Query: Retrieve metadata and balances for any token created by the factory.
  • User Dashboard: View the latest 10 tokens created by the user, with one-click address copying.
  • Shibarium Efficiency: Low-cost transactions and fast confirmations on Shibarium.
  • Security: Non-custodial, with all operations executed via MetaMask.

Future Optimizations

To enhance Woofswap Token Factory, we propose the following improvements:

  • Token Search: Add filtering and search for created tokens by name or symbol.
  • Event Notifications: Implement real-time alerts for token creation and transactions.
  • Multi-Language Support: Localize the interface for global accessibility.
  • Explorer Integration: Link token addresses to Shibariumscan for transparent transaction tracking.
